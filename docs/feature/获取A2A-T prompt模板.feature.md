# 获取A2A-T prompt模板

## 需求背景

A2A(Agent to Agent)协议是由Google等发起的开放标准，旨在实现不同技术框架或供应商构建的AI智能体(Agent)之间进行安全、跨平台的通信和写作。

为满足电信领域复杂的网络管理、业务编排与故障自愈等场景需求，需在通用A2A协议基础上，设计并实现面向电信领域的专用扩展协议-A2A-T(Agent to Agent for Telecom)。

A2A-T继承A2A协议的核心交互框架，支持智能体之间的发现、协作与任务委托，同时针对电信领域业务进行专项增强。

## 需求描述

使用python加载A2A-T prompt加载模块，同时支持从本地文件获取和在线方式获取。

## 需求要求

1. 在线方式获取，首次获取到后需要存放到本地，避免后续重复访问网络进行获取。
2. 在线获取需要支持url获取和访问Agent的IP:PORT获取
3. 支持多语言的prompt模板加载
